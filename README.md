# 확장형 게시판
기본 CRUD 게시판에 기능을 붙이는 확장형 게시판입니다.


### 사용한 것
Java, SpringBoot, Spring Security, JPA, H2, Thymeleaf, JavaScript, CSS, Bootstrap


### 구현된 기능
- 로그인
- 회원가입
- 질문 CRUD
- 답변 CRUD
- 추천기능
- 앵커기능
- 마크다운
- 검색
- 마이페이지 : 내가 쓴 글, 내가 쓴 답변
- 조회수

### 구현할 기능
- 메인페이지
- 공지사항 
- 파일첨부 
- 즐겨찾기 
- 소셜 로그인 
- 답변 페이징 및 정렬
